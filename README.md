# Hotel Booking Cancellation Prediction

## Problem
Predict whether a hotel booking will be canceled.

## Workflow
- Data Analysis (EDA)
- Logistic Regression (baseline)
- Random Forest (improvement)
- XGBoost (final model)

## Key Insights
- High lead time → higher cancellation
- Repeat customers cancel less
- Engagement reduces cancellation

## Results
- Logistic Regression: ~79% accuracy
- XGBoost: ~87% accuracy with better recall

## Files
- Sprinting DS → EDA
- LogReg + RF → baseline models
